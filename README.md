I am porting Syobon no Action (Syobon action/Cat Mario) to JS+WASM.  
I am using assets and code from the original source.  

The model I used for this porting project is Gemini.  

Note that I have made some modifications to the original.  
For example, the parts that depended on DxLib and the ending credits (content).  

The original README is in "りーどみー_en.txt".  
I had Gemini translate the "readme.txt" file into other languages on its own.  
※The games are not translated at all, so plesae be careful.  

Original author: tiku（ちく）
